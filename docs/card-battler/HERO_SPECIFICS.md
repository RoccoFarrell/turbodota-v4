# Hero-Specific Card Designs

This document contains the complete card designs for all Dota 2 heroes across all 4 rarities (COMMON, UNCOMMON, RARE, LEGENDARY).

## Design Pattern

- **COMMON**: Just attack or block, no special effects
- **UNCOMMON**: Gains weak version of hero's unique power
- **RARE**: Stronger hero power + 50% stat boost
- **LEGENDARY**: Strongest hero power + 100% stat boost + cost reduction

## Notes

This document contains card designs for **all 127 Dota 2 heroes** across all 4 rarities (COMMON, UNCOMMON, RARE, LEGENDARY). Each hero follows the same pattern:

1. **COMMON**: Base stats only, no special effects
2. **UNCOMMON**: Weak hero power added
3. **RARE**: Stronger hero power + 50% stat boost
4. **LEGENDARY**: Strongest hero power + 100% stat boost + cost reduction

Heroes are organized by primary attribute (Strength, Agility, Intelligence, Universal) based on the [Liquipedia Dota 2 Heroes Portal](https://liquipedia.net/dota2/Portal:Heroes).

**Total Heroes**: 127 heroes  4 rarities = 508 card definitions

**Hero Distribution**:
- Strength: 36 heroes
- Agility: 35 heroes
- Intelligence: 34 heroes
- Universal: 22 heroes
